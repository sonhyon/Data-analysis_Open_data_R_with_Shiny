HTTP는 데이터를 암호화하지 않고 HTTPS는 HTTP의 보안 버전이다.
HTTPS는 암호화해서 데이터를 보내는데 API에게 url을 보냈을 때 HTTPS였어서 데이터가 제대로 안받아들여졌음
